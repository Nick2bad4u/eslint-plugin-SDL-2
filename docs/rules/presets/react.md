# ⚛️ React

Use for JSX parser compatibility in React/TSX projects. This preset primarily
configures JSX parsing context.

## Config key

```ts
sdl.configs.react;
```

## Flat Config example

```ts
import sdl from "eslint-plugin-sdl-2";

export default [...sdl.configs.react];
```
